#  Notifications

## Delete user notifications

**DELETE** /api/v2.1/notifications/

**Sample request**

```
curl --request DELETE 'https://cloud.seatable.io/api/v2.1/notifications/' \
--header 'Authorization: Token 60ba962845e8380d25948b95c3c439165fafbff1'

```

**Sample response**

```
{
    "success": true
}

```
