# Ping

## Ping

Use this simple request to check the status of the dtable-server.


**URL Structure**

> **\[GET]** /dtable-server/ping/


**Request Authentication**

> None




**Sample Request**

> ```
> curl 'https://cloud.seatable.io/dtable-server/ping/'
> ```

**Sample Response (200)**

> ```
> pong
> ```
